-= SD card boot image =-

Platform: zc702
Application: /wrk/hdstaff/xd-relman/work/XIDANE_DEV/regression/products/xidane/samples/file_io_manr_sobel_standalone/TEST_WORK_lnx64/file_io_manr_sobel_standalone/manr_sobel_fio.elf

1. Copy the contents of this directory to an SD card
2. Set boot mode to SD
   Revision C and earlier boards:
     Jumper J22 1-2
     Jumper J25 1-2
   Revision D and newer boards:
     DIP switch SW16 positions 3 and 4 set to 1
3. Insert SD card and turn board on
